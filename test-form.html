Google Workspace Spam Filtering:

  • Check Spam folder in Gmail
  • Check All Mail folder
  • Search Gmail for: from:onboarding@elystra.online

Domain Authentication Required:

Google Workspace blocks emails from unverified domains. You need:

  • SPF Record: v=spf1 include:sendgrid.net ~all
  • DKIM Records: From SendGrid dashboard
  • Domain Authentication: In SendGrid settings
